Contact info widget with Google map

== Description ==

Show your customers how to get to your business with this contact info widget that includes an interactive Google map.

Display the following optional information in a widget

* Interactive Google map
* Customizable title
* Business name
* Address
* Phone number
* Email address

You can also change certain settings on the map like

* Size
* Zoom
* Color

Learn more at http://www.kinwebdesign.com

== Installation ==

1. Upload the folder, 'kin-visitantes' and all its files to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
